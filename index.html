E-Commerce Marketplace API arrow_outward

I collaborated on this project to build the backend of an online ecommerce marketplace where users sign up as buyers or vendors of products. I was responsible for implementing login and authentication, user-vendor-admin roles and permissions, email verification and password reset, product CRUDs, bank account verification with Flutterwave SDK, payment integration with Flutterwave gateway and order-payment linkage, file upload with Multer, Cloudinary and multer-storage-cloudinary, search, indexing and pagination on the backend, and more...

Chat Application API arrow_outward

The deBeeChat API empowers developers to access, manage, and interact with user, post, comment and like resources. This includes functionalities such as user registration, login, post creation, commenting on posts, replying to comments, and liking posts and comments. Certain endpoints require authentication before performing any CRUD operation.

Movie App arrow_outward

A Responsive Movie App built with HTML, Vanilla CSS and Vanilla Javascript. The app fetches data of movies from an API. Another API allows users interact by liking and commenting on the movies. With the browser's local storage, a movie can only be liked once. Dark mode (which is the default design) and light mode settings are available.
